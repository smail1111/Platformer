# Platformer

This is my second mostly original project.

I will be making a platformer in pygame.

## How To Run

1. Clone Repository

`gh repo clone smail1111/Platformer`

2. Run `main.py`

`python3 main.py`
* **The game will not run on python 3.14. Python 3.13 is required for pygame to work.**

## Game Info

1. Controls

Move Left/Right -> Arrow Keys L/R

Sprint -> LShift

Jump -> Space

Exit Level -> Escape

2. Levels

There are currently five levels you can play, each with their own mechanics.
I might add more levels in the future with new mechanics,
but that's probably going to be the entire game.
