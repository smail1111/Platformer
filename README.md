# Platformer

This is my second mostly original project.

I will be making a platformer in pygame.

## How To Run

1. Clone Repository

`git repo clone smail1111/Platformer`

2. Run `main.py`

`python3 main.py`
* **The game will not run on python 3.14. Python 3.13 is required for pygame to work.**

## Game Info

1. Controls

Move Left/Right -> Arrow Keys L/R

Sprint -> LShift

Jump -> Space
